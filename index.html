Settings

Stored only in this browser's localStorage. Calls go directly from your browser to the Claude API. Get a key at console.anthropic.com.

Works with Ollama, vLLM, LiteLLM or any OpenAI-compatible server. For Ollama, start it with OLLAMA_ORIGINS=* so the browser can reach it, and set a large context window (num_ctx) — repository-wide questions send long report texts.

Built-in API via proxy uses a server-side demo key managed by the app owner. It is intended for portfolio review, has rate and spend limits, and may be temporarily unavailable. No API key is stored in this browser.

When the hosted proxy is not deployed on this origin, this option will show a request error. You can still use your own Claude key or an OpenAI-compatible local endpoint.

Token usage & cost

Exact token counts from API responses, priced at claude-sonnet-4-6 list rates ($3 / $15 per 1M in/out; caching $3.75 write / $0.30 read). Open-source local traffic shows what the same tokens would have cost on Claude; built-in proxy traffic is subject to demo limits.
