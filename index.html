Settings

Stored only in this browser's localStorage. Calls go directly from your browser to the Claude API. Get a key at console.anthropic.com.

Works with Ollama, vLLM, LiteLLM or any OpenAI-compatible server. For Ollama, start it with OLLAMA_ORIGINS=* so the browser can reach it, and set a large context window (num_ctx) — repository-wide questions send long report texts.

Built-in API via proxy is the default portfolio-demo mode. It uses Claude Haiku through a server-side key managed by the app owner, so no API key is stored in this browser.

This low-cost Haiku setup has rate and spend limits, may be temporarily unavailable, and can be less capable on complex synthesis. Use your own Claude key or an OpenAI-compatible local/open-source endpoint for higher limits, stronger models, or private demos.
